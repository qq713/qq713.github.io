# 图片压缩网站

一个简洁优雅的在线图片压缩工具，采用苹果设计风格。

## 功能特点

- 支持上传 PNG、JPG 等格式的图片
- 可自定义压缩比例
- 实时预览压缩前后的图片效果
- 显示压缩前后的文件大小
- 支持下载压缩后的图片
- 响应式设计，支持各种设备
